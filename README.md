# 🦅 NORZEC | Inteligência Epidemiológica: O Custo Invisível do Carvão
*"Matematizando incertezas."*


Este repositório contém o painel interativo de **Saúde Pública e Epidemiologia** desenvolvido pela **NORZEC - Inteligência Analítica**. O projeto demonstra a aplicação de Data Science e Data Storytelling para mapear o impacto da atividade mineral na saúde das comunidades locais em Moatize, Tete (Moçambique).

## 🎯 Visão Estratégica do Projeto

O debate sobre a mineração a céu aberto frequentemente carece de evidências quantificáveis. A NORZEC desenvolveu este dashboard para traduzir dados ambientais difusos em **inteligência acionável para o Estado e ONGs**.

O painel cruza duas variáveis fundamentais:
1. **Níveis de Poeira (PM10):** Termodinâmica ambiental oriunda da extração de carvão.
2. **Admissões Pediátricas:** Crises respiratórias e asma no Hospital Distrital de Moatize.

### O Problema (O Caos Analítico)
ONGs e instituições de saúde acumulam anos de dados de campo (inquéritos, admissões hospitalares) que morrem em planilhas de Excel. Sem cruzamento estatístico e visualização limpa, não há evidência. Sem evidência, não há renovação de financiamento (USAID, Banco Mundial, OMS).

### A Solução (Imposição NORZEC)
Transformar dados brutos num painel executivo onde a "carga cognitiva" é zero. A correlação entre o aumento de partículas PM10 na época seca e o pico de +312% nos internamentos infantis é demonstrada de forma irrefutável num único ecrã.


## 💻 Tecnologias e Metodologia

Para garantir performance imediata e acessibilidade em qualquer dispositivo, a interface foi construída com tecnologias web modernas, sem a necessidade de softwares pesados do lado do cliente:

* **HTML5 & CSS3:** Estruturação semântica.
* **Tailwind CSS:** Estilização utilitária para garantir um "Dark Mode" premium e responsivo (redução de fadiga visual).
* **JavaScript (Vanilla):** Lógica de processamento e interatividade no *client-side*.
* **Chart.js:** Renderização do gráfico de duplo eixo (Causa vs. Efeito) focado em *Data Storytelling* (cores de alerta apenas nos dados críticos).
